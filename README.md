🏦 Smart Banking System with AI Fraud Detection

A full-stack banking application built using Python and Streamlit that simulates real-world banking operations with secure authentication, transaction management, and machine learning-based fraud detection.

🚀 Features
🔐 Authentication & Security
Secure login system with hashed PIN (SHA-256)
Input validation for account creation
Protected user data storage
💰 Banking Operations
Deposit and withdraw money
Real-time balance updates
Fund transfer between accounts
Prevents self-transfer
🔄 Smart Transfer System
UPI-style receiver name preview
Transaction validation and error handling
🤖 AI Fraud Detection
Integrated Isolation Forest
Detects abnormal transactions based on user history
Real-time fraud alerts
📊 Analytics & Visualization
Transaction history tracking
Interactive graphs using Pandas
Mini statement on dashboard
📄 Bank Statement (PDF)
Generate downloadable PDF statements
Includes:
User details
Account balance
Transaction history
👤 Profile Management
Update name, email, and PIN
Secure PIN re-hashing
Delete account feature
🛠 Tech Stack
Frontend: Streamlit
Backend: Python (OOP)
Database: JSON
Machine Learning: Scikit-learn (Isolation Forest)
Data Handling: Pandas
PDF Generation: ReportLab
